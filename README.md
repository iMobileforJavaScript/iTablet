# iTablet
SuperMap App


### 目录结构
> * Android         (Android原生工程目录)
> * Ios             (ios原生工程目录)
> * node_modules    (package.json文件声明的依赖)
> * ackage.json     (npm的管理包的文件)
> * App.js          (app的入口文件)
> * Index.js        (打包运行的入口文件)
> * Src             (目录入口文件)
> * _tests_         (测试文件)
